# 1. 컴프리헨션
 - 딕셔너리 컴프리헨션
  - ```word = letters```
  - ```letter_counts = {letter:word.count(letter) for letter in word}```
  - 위와 같이 했을 때 key 중복된 값을 제외하고 갯수를 vaule로 나타낸다
  - 딕셔너리 컴프리헨션으로 사용하면 정렬은 계속해서 바뀐다
